# 📊 AQI Index Analysis Dashboard

## 📝 Project Overview
The **AQI Index Analysis Dashboard** is an interactive Power BI project designed to analyze and visualize air quality trends across different states and cities in India.  

This dashboard provides meaningful insights into pollution levels, major pollutants, AQI categories, and air quality trends over time, helping users better understand environmental conditions through data visualization.

---

## 🎯 Objective
The objective of this project is to:
- Monitor and analyze Air Quality Index (AQI) levels
- Identify highly polluted cities and states
- Understand the occurrence of major pollutants
- Visualize AQI trends over time
- Support environmental awareness through data-driven insights

---

## 📌 Key Features
- 🌍 **State-wise AQI Analysis** using map visualization
- 🏙️ **City-wise AQI Comparison**
- 📈 **AQI Trend Analysis Over Time**
- 🧪 **Primary Pollutant Identification**
- 📊 **AQI Level Distribution**
- 🔎 Interactive filters for:
  - Year
  - State
  - City

---

## 📂 Dataset Information
The dashboard uses an Excel dataset containing:
- Date/Year
- State
- City
- AQI Value
- AQI Category
- Primary Pollutant
- Pollutant Frequency Data

---

## 🛠️ Tools & Technologies Used
- **Power BI** – Dashboard Development & Visualization
- **Microsoft Excel** – Data Source
- **Power Query** – Data Cleaning & Transformation
- **DAX** – KPI and Measure Calculations
